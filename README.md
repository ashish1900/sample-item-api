# Sample Item Management REST API

This is a simple Spring Boot REST API developed as part of a sample task.

## Features
- Add a new item
- Get item by ID
- Input validation for required fields
- In-memory data storage

## Tech Stack
- Java 17
- Spring Boot
- REST API
- Docker

## API Endpoints
- POST /items → Add new item
- GET /items/{id} → Get item by ID
- GET /items/ping → Health check
